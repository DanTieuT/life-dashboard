⚡ Setup Your Dashboard

This takes about 5 minutes. You only do it once — after that, just sign in with Google on any device and everything syncs automatically.

1

Create a free Firebase project

Go to console.firebase.google.com, click "Add project", give it a name like "My Dashboard", and click through (you can disable Google Analytics).

2

Add a Web App

In your new project, click the </> Web icon. Give the app a nickname (e.g. "Dashboard"), click "Register app". You'll see a firebaseConfig object — copy it.

3

Enable Firestore + Google Sign-In

In the left sidebar, go to Build → Firestore Database → Create database → Start in test mode. Then go to Build → Authentication → Get started → Google → Enable it.

4

Paste your config below

Paste the entire firebaseConfig = { ... } block (or just the JSON object) below and click Connect.
